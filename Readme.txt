Descargar FFmpeg >>> para poder pasar de formato video a mp3
este se descarga en https://ffmpeg.org/download.html . 
1) ir a en enlace de version windows "Windows builds from gyan.dev"
2) buscar el link de git y descargar
3) descomprimir y pasar el contenido a una carpeta nueva "FFMPEG" dentro del disco local
4) entrar en la carpeta bin y copiar la direccion de la carpeta 
5) abrir las variables de entorno y dentro de path agregar esta ruta. y listo

pygame              >>>> para poder reproducir la musica de fondo 
urllib              >>>> para obtener la url de la musica 
speech_recognition  >>>> reconocimiento de voz
pyttsx3             >>>> para configurar la voz que usaremos como base
